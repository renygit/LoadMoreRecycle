# LoadMoreRecycle
只具有加载更多的RecycleView，下拉刷新请使用其它库。可以设置加载中样式，加载失败，没有更多。

引用
===========================
```Java
maven { url "https://jitpack.io" }
```Java
  dependencies {<br>
	        compile 'com.github.renygit:LoadMoreRecycle:0.0.1'<br>
	}<br>
